

 ###                     `Getting Started`
 ## 1-Fork and Clone the farmers-market-api repo 
 ## 2-Follow ReadMe instructions for the farmers-market-api repo.

 ## 4-Scroll through the list of farmers markets or click on New Market to create a new one.
